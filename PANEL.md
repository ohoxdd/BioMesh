# PANEL DE CONTROL 

Este documento describe el los contenidos que debe tener el panel de control del proyecto BioMeshP2P. El panel de control es una interfaz web que permite visualizar la información recopilada por los nodos emisores.Cualquier contradiccion a este documento, gana este documento.

## Contexto y proposito del Panel de Control

Dados los datos que emiten los nodos emisores, este panel debe ser capaz de representarlos de manera interactiva y visual. El usuario observador podra consultar los datos de los distintos nodos a los que tiene acceso. 

## Funcionamiento

El usuario selecciona un pin de uno de los nodos emisores, y el panel muestra un conjunto de graficas representativas de las metricas que mide el nodo emisor.
